## Tic Tac Toe

This is a tic tac toe game handbuilt by humans.

Use "git clone https://github.com/flatspider/tic-tac-toe-fractal.git" to install on your computer.

And then run "npm install" and then "npm run dev" to run locally.

The project should then be visible at "http://localhost:5173/"

Have fun!
